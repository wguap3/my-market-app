# ShopFront

Веб-приложение «Витрина интернет-магазина»

## Возможности

- Просмотр товаров с поиском, сортировкой и пагинацией
- Страница отдельного товара
- Корзина: добавление, изменение количества, удаление товаров
- Оформление заказа и просмотр истории заказов

## Стек технологий

- **Java 21**
- **Spring Boot 3**
- **Spring Web MVC**
- **Spring Data JPA / Hibernate**
- **Thymeleaf**
- **PostgreSQL** (H2 для тестов)
- **Maven**
- **Docker**
- **JUnit 5 / Spring Boot Test**

## Запуск

Убедитесь, что установлен Docker и Docker Desktop запущен.

**1. Клонируйте репозиторий:**

```bash
git clone https://github.com/wguap3/my-market-app.git
cd my-market-app
```

**2. Соберите проект:**

```bash
mvn clean package -DskipTests
```

**3. Запустите контейнеры:**

```bash
docker-compose up --build
```

Приложение будет доступно по адресу: [http://localhost:8080](http://localhost:8080)

## Запуск тестов

```bash
mvn test
```

## Эндпоинты

### Товары

| Метод | Endpoint | Описание |
|-------|----------|----------|
| `GET` | `/items` | Витрина товаров (поиск, сортировка, пагинация) |
| `POST` | `/items` | Изменить количество товара в корзине |
| `GET` | `/items/{id}` | Страница товара |
| `POST` | `/items/{id}` | Изменить количество товара в корзине |

### Корзина

| Метод | Endpoint | Описание |
|-------|----------|----------|
| `GET` | `/cart/items` | Страница корзины |
| `POST` | `/cart/items` | Изменить/удалить товар в корзине |

### Заказы

| Метод | Endpoint | Описание |
|-------|----------|----------|
| `GET` | `/orders` | Список всех заказов |
| `GET` | `/orders/{id}` | Страница заказа |
| `POST` | `/orders/buy` | Оформить заказ |
